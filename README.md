# 🌌 침몽도시: 루시드 다이버 (Lucid Diver)

> **“현실의 법칙이 붕괴한 꿈의 도시, 그 속에 갇힌 유물을 회수하고 각성하라!”**
>
> 《침몽도시: 루시드 다이버》는 현실과 왜곡된 꿈이 겹친 가상 공간 '침몽도시'를 배경으로 하는 **2.5D 라이트 PvE 익스트랙션 액션 RPG** 프로젝트입니다. 
> 플레이어는 특수 자각몽 탐사자인 '루시드 다이버'가 되어, 꿈속에 들어가 일상 기물을 즉석 무장으로 삼아 괴이를 퇴치하고 고가치의 '심상핵'을 회수하여 현실로 탈출해야 합니다.

---

## 🚀 프로젝트 개요

- **장르**: 2.5D 라이트 PvE 익스트랙션 액션 RPG
- **플랫폼**: PC (Windows) / Steam 타겟
- **기술 스택**: Unity 6, 3D HDRP/URP, 2D Spine/Sprite Animation, Photon PUN2, C#
- **핵심 슬로건**: *“현실의 무기는 멈춘다. 오직 왜곡된 꿈의 기물만이 길을 연다.”*

---

## 🌟 핵심 게임 피처

1. **2.5D 비주얼 스타일**
   - 3D로 정교하게 입체화된 공간 레벨 위에 2D 서브컬처 스타일 캐릭터 스프라이트를 얹은 고품질 하이브리드 비주얼.
2. **드림 포지드 (Dream-Forged) 즉석 무장**
   - 꿈속에 들어서면 일반 총기나 칼은 약화됩니다. 필드에서 획득한 깨진 우산, 금 간 거울, 거꾸로 가는 시계 등 초현실적인 '심상기물'을 즉석에서 기괴하고 강력한 무기로 실시간 변형시켜 싸워야 합니다.
3. **1+2+1 전략적 파티 시스템**
   - 플레이어가 직접 조작하는 **1명의 메인 다이버**, 위기 상황에 2D 컷인 연출과 함께 소환되는 **2명의 스트라이커 지원 캐릭터**, 현실 거점에서 정보와 안정화 버프를 주는 **1명의 오퍼레이터**의 유기적 편성.
4. **라이트 익스트랙션 (PvE Extraction)**
   - 플레이어 간 약탈(PvP)과 영구 사망의 가혹함을 배제하고, 사망(강제 각성) 시 아이템 일부 소실 규칙과 **4칸 그리드 안전 보존 슬롯**만을 적용하여 진입장벽을 낮춘 캐주얼한 파밍 및 탈출 재미.
5. **협동 요청 핑 시스템**
   - 세션 초반에는 각자 외곽에서 파밍을 진행하다가, 중심부 고위험 구역의 보스 룸에 진입할 때 핑을 보내 세션 내 다른 플레이어들과 함께 협동 레이드를 공략하고 개별 루팅 보상을 획득하는 구조.

## 🤖 Antigravity Co-PM 협업 규칙

본 저장소의 문서는 AI 에이전트(Antigravity)를 Co-PM 삼아 공동 관리합니다.

1. **AI 초안 작성**: AI가 문서를 작성하면 최상단에 `[!IMPORTANT]` 배너가 삽입되어 초안(참고용)임을 명시합니다.
2. **PM 검수 및 확정**: 기획자/PM이 초안을 검토하고, 사양이 최종 승인되면 배너를 제거하여 **'확정 사양'**으로 전환합니다.
3. **히스토리 작성**: 사양이 변경될 때마다 각 문서 하단의 `## 📜 Revision History` 테이블에 버전 이력과 상세 수정 내역을 불렛 포인트로 기록합니다.
